## ✨ ฟีเจอร์ทั้งหมด (Features)

### 🏠 โรงรถ & ตัวรถ
- **ซื้อโรงรถจากโรบิน** — เปิดเมนูก่อสร้าง เลื่อนเลือกได้ **2 รุ่น**:
  - 🔵 **BMW S1000RR** (สีดำ)
  - 🔴 **Ducati V4S** (สีแดง)
- เลือกรุ่น **ตั้งแต่ตอนสั่งสร้าง** แล้ววางตำแหน่งได้ตามปกติ
- **สร้างได้หลายคัน** — แต่ละหลังเลือกรุ่นแยกกัน ตั้งชื่อแยกกันได้
- รถ **จอดโชว์ในโรงรถ** และเดินเข้าโรงรถได้แม้ยังไม่ขึ้นรถ
- ขี่ควบคู่กับม้าได้ (คนละระบบ) — เรียก/ขี่/ผูกกับผู้เล่นเหมือนม้า

### ⚙️ ระบบเกียร์อัตโนมัติ (แบบเกมแข่งรถ)
- **เกียร์ 1 → 6 เปลี่ยนเอง** ไล่เสียงต่อเนื่องตอนขี่ ไม่ต้องกดปุ่มเกียร์
- **ความเร็วเพิ่มขึ้นตามเกียร์** อัตโนมัติ (เกียร์ 1 = ช้าสุด → เกียร์ 6 = เร็วสุด)
- หยุดรถ → กลับเกียร์ 1 + ความเร็วต่ำสุดเสมอ
- เกียร์ 6 วนเสียงต่อเนื่องสำหรับขับค้างนานๆ

### 🔊 เสียงเครื่องยนต์
- เสียงสตาร์ทมือ → เดินเบา → ไล่เกียร์ พร้อมเสียงจริงจากซูเปอร์ไบค์
- **เบิ้ลเครื่อง** — กดปุ่มค้างเพื่อเร่งเครื่องอยู่กับที่ ปล่อยแล้วเสียงหรี่ลง
- ปิดเสียงกีบม้า ใช้เสียงเครื่องยนต์แทน
- **มัลติเพลเยอร์**: ทุกคนในแมพเดียวกันได้ยินเสียงเครื่องเหมือนกัน (ดังตามระยะ)

### 🧑‍🤝‍🧑 คนซ้อนท้าย (Multiplayer)
- เพื่อนเดินมาข้างรถ **กดปุ่ม action ขึ้นซ้อนท้าย** ได้
- คนซ้อนนั่งเกาะรถทุกทิศ + มีอนิเมชันกระโดดขึ้น/ลง (เห็นทุกจอ)
- คนขับข้ามแมพ → คนซ้อน **วาร์ปตามอัตโนมัติ** (มีเฟดจอดำ)
- กดปุ่ม action อีกครั้งเพื่อลง

### 🎛️ อื่นๆ
- **HUD วงกลมบอกเกียร์** มุมบนซ้าย (เห็นทั้งคนขับและคนซ้อน)
- ควันท่อไอเสียตอนวิ่ง
- รองรับ **คีย์บอร์ดและจอย** ทุกปุ่ม
- ข้อความสองภาษา **ไทย / อังกฤษ**

## 📥 การติดตั้ง (Installation)

1. ติดตั้ง [SMAPI](https://smapi.io) (เวอร์ชัน 4.0.0 ขึ้นไป)
2. โหลด [Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098) (แนะนำสำหรับการตั้งค่า)
3. แตกไฟล์ mod ลงในโฟลเดอร์ `Mods`
4. เปิดเกมผ่าน SMAPI

## 🎮 วิธีเล่น (How to Use)

| การกระทำ | วิธี |
|---|---|
| **สร้างโรงรถ** | คุยกับโรบิน → เมนูก่อสร้าง → เลื่อนหา "โรงรถ BMW/Ducati" → สร้าง |
| **ขี่รถ** | เดินไปคลิกที่รถ (เหมือนขี่ม้า) |
| **เปลี่ยนเกียร์** | อัตโนมัติ — แค่ขี่ไปเรื่อยๆ เกียร์เพิ่มเอง |
| **เบิ้ลเครื่อง** | กดปุ่มเบิ้ลค้าง (ค่าเริ่มต้น `R` / R-Stick) |
| **ดับเครื่อง** | กดปุ่มดับ (ค่าเริ่มต้น `K` / Back) |
| **ขึ้นซ้อนท้าย** | เดินไปข้างรถที่มีคนขับ กดปุ่ม action |
| **ลงจากซ้อน** | กดปุ่ม action อีกครั้ง |

## ⚙️ การตั้งค่า (Configuration — ผ่าน GMCM)

| ตัวเลือก | ค่าเริ่มต้น | คำอธิบาย |
|---|---|---|
| Build cost | 10,000 G | ราคาสร้างโรงรถ |
| Materials | ไม้ / หิน (ปรับเป็นเหล็ก/อีรีเดียม/แบตได้) | วัสดุที่ใช้สร้าง |
| Build days | 2 | จำนวนวันที่โรบินสร้าง |
| Min speed | 1.5 | ความเร็วเกียร์ 1 |
| Max speed | 8 | ความเร็วเกียร์ 6 |
| Engine volume | 1.0 | ระดับเสียงเครื่อง (0–2 เท่า) |
| Engine-off / Rev keys | K / R | ปุ่มดับเครื่อง / เบิ้ล (ตั้งเองได้ ทั้งคีย์บอร์ด+จอย) |
